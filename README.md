# ApkInstaller
### 这是当初自己写着玩的一个APK安装软件，原理很简单，就是利用Shell进行安装。
## 目前实现了以下功能：
- APK安装卸载
- 路径显示
- 源文件删除
- 夜间模式
- APK权限与Activity显示
- 兼容AndroidQ设备
